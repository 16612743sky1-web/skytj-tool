# 펫포커스 재고/원가 관리 — 운영 지침

> 대상: 대표(고윤상) 및 공동 작업자 · 사업자: 펫포커스 (일반과세자, 709-39-01334)
> 도구: `petfocus/index.html` (핸드폰 브라우저용 웹앱)

---

## 1. 이 도구가 하는 일 / 안 하는 일

**하는 일**
- 원재료(우피·식초·스틱·포장지) 및 완제품(우피껌) 재고 실시간 파악
- 생산 배치 입력 시 총원가·개당 원가·수율(%) 자동 계산
- 판매 입력 시 채널 수수료 차감 후 순매출·한계이익 자동 계산
- 월별 요약: 매입액 / 생산원가 / 순매출 / 한계이익

**안 하는 일 (별도 관리 필요)**
- 부가세 신고 자료 자체를 만들지 않음 → 홈택스/세무대리인 기준은 **적격 증빙 원본**
- 임차료·통신비 등 배치와 무관한 고정비 배부 (2단계 로드맵)

## 2. 입력 규칙 — 이것만 지키면 숫자가 맞습니다

1. **돈이나 물건이 움직이면 그날 입력한다.** 몰아서 입력하면 수율·재고가 틀어집니다.
2. **입력 전 작성자 버튼(✍️)을 자기 이름으로 선택한다.** 누가 입력했는지가 기록됩니다.
3. **매입은 증빙 종류를 반드시 선택한다.** '증빙 없음'은 부가세 매입세액 공제가 안 되는 돈입니다. 월말에 '증빙 없음' 건수를 확인하고 0건을 목표로 합니다.
4. **폐기·시식 증정·실사 차이는 '조정'으로 기록한다.** 사유 없는 조정은 저장되지 않습니다. 조정 기록이 없으면 수율이 실제보다 좋게 보이는 착시가 생깁니다.
5. **생산 입력 시 '완성 중량(kg)'을 가능하면 함께 입력한다.** 우피는 건조 수축이 크므로 kg 기준 수율(완성kg ÷ 투입kg)이 원물 매입 판단의 핵심 지표입니다.
6. **작업 인시(인×시간)에는 대표 본인의 시간도 포함한다.** 내 인건비를 0으로 잡으면 개당 원가가 과소계상되고, B2B 견적에서 손해 보는 가격이 나옵니다.

- 분기 1회 이상: 실제 전기요금 고지서 ÷ 건조기 가동시간으로 시간당 전기료를 재검증합니다.

## 4. 두 사람이 같이 쓰는 방법

**방법 A — 간단 (백업 파일 공유)**: 설정 → 파일 내보내기 → 카톡 전송 → 상대방이 파일 가져오기. 기록은 ID 기준으로 병합되므로 중복되지 않습니다. 주 1회 이상 권장.

**방법 B — 자동 동기화 (권장, 최초 1회 5분 설정)**
1. https://console.firebase.google.com 에서 프로젝트 생성 (무료 Spark 요금제면 충분)
2. 빌드 → Realtime Database → 데이터베이스 만들기 (지역: 아무거나)
3. 규칙 탭에서 아래처럼 설정 후 게시:
   ```json
   { "rules": { "petfocus-여기에-아무도-못맞출-긴-문자열": { ".read": true, ".write": true } } }
   ```
4. 두 핸드폰의 설정 탭 '동기화 주소'에 똑같이 입력:
   `https://프로젝트명-default-rtdb.firebaseio.com/petfocus-여기에-아무도-못맞출-긴-문자열.json`
5. 이후 저장할 때마다 자동 동기화. 상단 초록 점 = 정상.

⚠️ 주소가 곧 비밀번호입니다. 경로 문자열을 길고 무작위하게 만들고, 카톡 외 공개된 곳에 올리지 마십시오. 고객 개인정보(주소·전화번호)는 이 앱에 입력하지 않습니다.

## 5. 백업 및 리스크 통제

- **월 1회 (부가세 대비)**: 파일 내보내기 → 구글 드라이브 보관. 파일명에 날짜가 자동으로 들어갑니다.
- 핸드폰 분실/교체 시: 새 폰에서 URL 접속 → 백업 파일 가져오기 (또는 동기화 주소 입력).
- 브라우저 데이터 삭제 시 로컬 기록이 사라집니다. 동기화(방법 B)를 쓰면 복구 가능.
- 이 앱의 숫자는 **관리회계용**입니다. 세무 신고는 세금계산서·카드전표 등 적격 증빙 원본이 기준이며, 앱의 매입액과 홈택스 매입 합계를 월말에 대사(눈으로 비교)하여 누락 매입을 찾아냅니다.

## 6. 가격 결정 기준 (계산 탭)

- **B2C 권장가** = 개당 원가 ÷ (1 − 목표 마진율). 목표 마진율 50% 미만으로 내려가면 채널 수수료·반품·광고비를 흡수하지 못합니다.
- **B2B 도매 공급가**: B2C가의 60~70% 선에서 시작하되, **개당 원가 + 최소 25% 마진**이 확보되지 않으면 수량이 아무리 커도 수락하지 않습니다.
- 수출(해외) 견적 시: 개당 원가에 박스·검역서류·물류비를 별도 가산한 뒤 마진을 계산합니다 (앱 원가에는 포함되어 있지 않음).
