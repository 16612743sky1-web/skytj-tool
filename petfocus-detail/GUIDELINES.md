# 펫포커스 상세페이지 생성기 · 사용 설명서

핸드폰이나 PC 브라우저에서 상품 정보를 넣으면, 상세페이지를 만들 **AI용 프롬프트**가 6단계로 자동으로 나오는 도구입니다.
상세페이지 이미지를 직접 그려주는 게 아니라, **AI(미리클·젠스파크·ChatGPT 등)에 붙여넣을 문장**을 만들어줍니다.

> 🔗 주소: https://16612743sky1-web.github.io/skytj-tool/petfocus-detail/

---

## 왜 이렇게 하나요? (외주 대신)

- 쿠팡 상세페이지 하나 = 외주 **20~80만원**, 게다가 **수정할 때마다 또 비용**.
- 이 도구로 만들면 제작·수정이 **무료·무제한**. 상품이 100개여도 외주비 0원.
- 대신 실제 이미지 생성은 무료 AI 툴(미리클/젠스파크/ChatGPT 등)에서 합니다. 이 도구는 **거기에 넣을 프롬프트까지** 담당합니다.

---

## ⭐ 가장 중요한 2가지 (꼭 기억)

### 1) 한글은 이미지에 넣지 않습니다
AI로 이미지를 만들 때 **이미지 안에 한글을 그려 넣으면 글자가 깨집니다**(외계어처럼). 이건 아무리 프롬프트를 잘 써도 못 막습니다.

**그래서 이렇게 합니다:**
1. 이미지는 **글자 없이**(제품컷·배경·무드컷만) 만든다 → 이 도구의 이미지 프롬프트엔 "글자 넣지 마"가 이미 들어 있음.
2. 한글은 **미리캔버스·캔바·파워포인트**에서 진짜 한글 폰트로 **따로 얹는다**.

→ 이렇게 하면 한글이 100% 선명하고, 나중에 글자만 고치기도 쉽습니다.

### 2) "AI 티"를 뺍니다
- **글**: "혁신적인·완벽한·최고의·특별한" 같은 티 나는 표현을 자동으로 금지하고, **등록한 브랜드 말투**를 유지합니다.
- **사진**: 완전히 새로 만든 사진은 AI 티가 납니다. 그래서 **찍은 사진을 보정**하는 걸 먼저 씁니다(실물이라 자연스러움).

---

## 준비물 (처음 한 번)

- 브랜드 한 줄 소개 / 강점 2~3개 / 주요 고객 / 브랜드 말투
- 쓸 AI 툴 하나 (무료로 시작 가능): 미리클, 젠스파크, ChatGPT 등
- 한글 얹을 툴 (무료): **미리캔버스** 추천

---

## 사용 순서

### 처음 세팅 — 딱 한 번
1. **1단계 브랜드 정보** 입력 → 자동 저장 (모든 상품에 재사용)
2. **3단계 지침** 확인 (기본값이 채워져 있음. 필요할 때만 수정)

### 신상품이 나올 때마다
1. 상단 **+ 새 상품** → **2단계 제품 정보** 입력 (제품명·성분·규격·가격·특징·급여법·주의사항)

### 상세페이지 만들기
1. **1·2·3단계 [복사]** → AI에 순서대로 붙여넣어 브랜드·제품·규칙을 이해시킴
2. **4단계 [복사]** → 상세페이지 **글(텍스트)부터** 완성
   - 맘에 안 들면 **4단계의 "수정 요청 프롬프트"** 로 몇 번이든 무료로 고침
3. **5단계**
   - **A 촬영본 살리기**: 찍은 사진을 올리고 이 프롬프트로 보정 (제일 자연스러움)
   - **B AI 실사 생성**: 직접 못 찍는 컷(급여 장면 등) 새로 생성
   - **C 상세페이지 조합**: 세로로 긴 상세페이지 이미지로 합치기 (글자 자리는 비워둠)
4. **6단계 안내대로** 미리캔버스에서 한글 카피를 얹고 → PNG로 저장 → **쿠팡 업로드**

---

## 한글 얹는 법 (미리캔버스 기준)

1. 미리캔버스 접속 → **새 디자인 · 상세페이지** 크기 선택
2. 5-C로 만든 **글자 없는 이미지**를 배경으로 업로드
3. **텍스트 추가** → 4단계에서 만든 카피를 섹션 위치에 붙여넣기
4. 폰트는 또렷한 한글: **Pretendard / 나눔스퀘어 / G마켓산스** 등
5. 제목 크게·본문 작게, 줄맞춤 정렬 → **다운로드(PNG)**

---

## 올리기 전 마지막 점검

- [ ] "혁신적·완벽·최고·특별한" 같은 **클리셰**가 남아있지 않은가
- [ ] 문장이 다 비슷한 길이로 반복되지 않는가 (길고 짧게 섞였는가)
- [ ] 사진 속 **발가락·글자·로고**가 이상하지 않은가
- [ ] 효능을 **단정**(예: "치석이 제거됩니다")하지 않았는가 → 반려동물 간식 표시광고 규정 주의
- [ ] 한글이 **깨지지 않고** 선명한가 (이미지 안이 아니라 얹은 글자인가)

---

## 데이터 / 저장

- 입력한 내용은 **이 브라우저에** 자동 저장됩니다(localStorage). 서버로 전송되지 않습니다.
- 다른 기기·다른 브라우저에서는 다시 입력해야 합니다(기기 간 동기화는 이 버전에 없음).
- 브라우저 데이터를 지우면 저장 내용도 사라지니, 중요한 카피는 따로 보관하세요.

## 자주 묻는 질문

**Q. 이 도구가 상세페이지 이미지를 바로 만들어 주나요?**
아니요. **프롬프트(AI에게 시킬 문장)**를 만들어 줍니다. 실제 이미지는 AI 툴에서 그 프롬프트로 만듭니다. 이렇게 나눠야 한글이 안 깨지고 수정이 무료입니다.

**Q. 사진이 아마추어인데 괜찮나요?**
네. 5단계 A(촬영본 보정)로 배경 정리·밝기 보정하면 훨씬 나아집니다. 못 찍는 컷만 B로 생성하세요.

**Q. 상품마다 톤이 달라지지 않나요?**
1단계 브랜드 말투를 한 번 등록하면 모든 상품이 같은 톤으로 나옵니다. 브랜드가 하나로 보이게 하려면 이게 중요합니다.

## 2026-07-17 업데이트 — 톤앤매너 유지 · 신제품 기능

### 🐶 퓨어 베이비 츄잉 불러오기 (홈)
홈 화면의 **"퓨어 베이비 츄잉 불러오기"** 버튼을 누르면 브랜드 정보(비어 있을 때)와
신제품 데이터(원재료·규격·특징·급여·전용 섹션 10개)가 자동으로 채워집니다.
**입력 0번으로 1~6단계 프롬프트가 완성**됩니다. 이미 만든 상품이 있으면 중복 생성되지 않습니다.

### 기존 상세페이지 톤앤매너 (3단계 · 공통, 한 번만)
기존 상세페이지의 구성·말투 요약이 기본값으로 들어 있습니다. 한 번 저장하면 모든 상품의
텍스트·이미지 프롬프트에 **"이 톤앤매너를 유지하고 상품 부분만 교체하라"**는 지시가 자동으로 붙습니다.
→ 신제품이 나와도 브랜드가 한 몸처럼 보입니다.

### 이 상품 전용 섹션 구성 (2단계 · 선택)
비우면 3단계 공통 섹션을 쓰고, 채우면 그 상품만 다른 구성을 씁니다.
(예: 퓨어 베이비 츄잉은 "첫 우피껌" 스토리 10섹션 전용 구성)

### 색감 콘셉트 (5단계 · 상품별)
- **크림·베이지 + 연한 민트** (추천 — 퍼피 라인, 크라프트 패키지와 어울림)
- **기존 톤 그대로** (하늘색 커버 + 옐로우·크림)

최종 결정자가 고를 수 있게 **두 색감으로 각각 프롬프트를 복사**해 결과물을 나란히 보여주세요.
색감을 바꾸면 5단계 프롬프트가 즉시 다시 생성됩니다.

### 📷 이미지 삽입 (5단계)
상세페이지에 쓸 사진을 도구 안에서 바로 등록: 드래그/선택 업로드, 설명 입력, ◀▶ 순서 이동,
교체·삭제. 상품별로 이 기기에 자동 저장됩니다(용량 자동 축소).

### 신상품이 나오면?
"+ 새 상품" → 2단계 제품 정보만 입력하면 끝. 브랜드·지침·톤앤매너는 그대로 재사용됩니다.
상품 고유 정보는 코드에 하드코딩하지 않는 것이 원칙입니다 (불러오기 버튼 1개만 예외).
