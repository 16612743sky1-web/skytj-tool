# 펫포커스 마케팅 (marketing)

펫포커스의 모든 판매 채널 문구·이미지 기획에 적용하는 기준.
상세페이지 프롬프트 생성기(`petfocus-detail/index.html`)와 같은 원칙을 공유한다.

## 판매 채널 (참고 원본)

- 쿠팡 브랜드 스토어: https://shop.coupang.com/vid/A01503210
- 네이버 스마트스토어: https://smartstore.naver.com/spdltusvpt
- 기존 상세페이지 원본: 저장소 `상세페이지(원본)/` (img_01~06)
- 기존 썸네일 원본: 저장소 `썸네일(원본)/`

## 브랜드 톤앤매너 (기존 상세페이지 기준 — 유지가 원칙)

- 말투: 과장 없이 다정한 존댓말. "~해 주세요", "~있어요". 보호자에게 친구처럼 설명.
- 구성 패턴: 메인 커버(큰 타이틀+실사+뱃지 4개) → "Check Point" 라벨+✔ 체크리스트
  → Point 01/02(차별점·무첨가) → 보호자 공감 인용구 → 실제 제품 사진(원물→손질→건조)
  → 크기·중량 안내 → 일일 권장 급여량 표 → ※ 유의사항 각주
- 색감: 크림·옐로우·베이지 배경, 여백 많게, 실제 제품 사진 중심 (과한 아이콘 금지)
- 신제품(퍼피 라인)만 연한 민트 포인트 컬러 허용 (퓨어베이베츄잉파일1.pdf 디자인 컨셉)
- 폰트: Pretendard Bold + Regular

## 반드시 지킬 규칙

1. **한글 깨짐 방지**: AI 이미지에 한글을 그려 넣지 않는다. 이미지는 글자 없이 만들고
   한글은 미리캔버스·캔바에서 별도 레이어로 얹는다.
2. **AI 티 제거**: "혁신적인·완벽한·최고의·궁극의" 같은 표현 금지. 문장 리듬 다양하게.
   구체적 수치·감각(식감·크기·냄새)으로 사실적으로. 촬영본 보정을 AI 생성보다 우선.
3. **효능 단정 금지**: "치석이 제거됩니다" 같은 질병 치료·예방 단정 표현 금지
   (표시광고 규제). "치석 관리에 도움" 수준의 사실 서술만.
4. **사진은 실제 촬영본 우선**: `pet focus 신제품이미/` 같은 실제 컷을 보정해 쓰고,
   못 찍는 컷만 AI 생성으로 보충.
5. **판매 구성: 모든 제품 단품 생산·판매** (2026-07 결정 — 세트 구성 없음.
   기존 상세페이지의 "세트구성으로만 판매" 문구는 과거 기준이니 새 작업에 복사 금지).
   용량은 추가 생산될 수 있으니 특정 용량을 단정하는 문구는 피한다.
6. **색감 등 디자인은 최종 결정자가 따로 있다**: 한 가지로 확정하지 말고
   추천안(크림·베이지+민트)과 대안(기존 하늘·옐로우)을 함께 만들어 비교로 제시.
   생성기 5단계의 색감 콘셉트 선택으로 두 벌 재생성 가능.

## 상품 정보 기준 (자주 쓰는 사실)

- 주력: 프리미엄 한우 우피껌 — 100% 국내산 한우 우피, 무첨가·무방부제·색소/향미제 무첨가, 국내 생산
- 기존 제품: 퓨어플레인츄잉(35g/70g), 닭안심 회오리츄잉(35g/70g), 건조닭안심(30g/50g)
- 신제품: 퓨어 베이비 츄잉 — 4개월 이상 퍼피용, 5~6cm 미니, 35g/200g,
  메인 메시지 "우리 아이가 처음 만나는 한우 우피껌" (첫 우피껌 포지셔닝)
- 규격 표기: 길이 12~15cm·둘레 2~3cm(성견용 스틱), 모양·크기가 달라도 중량 동일
- 급여 안내 표: 월령/체중별 권장량 + "충분한 물 제공, 보호자 관찰, 밀봉 보관"

## 산출물 체크리스트

- [ ] 브랜드 말투 유지되었는가 (기존 상세페이지 문장과 나란히 놓아도 어색하지 않은가)
- [ ] 금지 표현(AI 티·과장·효능 단정) 없는가
- [ ] 이미지 계획에 "글자 없이 생성 + 한글 별도 레이어" 명시했는가
- [ ] 수치·구성(중량, 개수, 세트)이 실제 상품 정보와 일치하는가
